# Expo HAS CHANGED

Read the exact versioned docs at https://docs.expo.dev/versions/v54.0.0/ before writing any code.

Note: this project targets **SDK 54** because that is the newest SDK the public App Store Expo Go currently supports (SDK 55's Expo Go is stuck in Apple review; 56/57 are TestFlight/simulator only). Do not bump the SDK to run in Expo Go — a higher SDK will fail with "incompatible version." Only move to 55+ when switching to a custom dev/production build (eas build / eas go).
